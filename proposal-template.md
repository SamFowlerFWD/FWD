### Technical Foundation

The build will be delivered to current technical standards:

- **Performance**: Page loads targeting Core Web Vitals compliance (LCP under 2.5 seconds, INP under 200ms)
- **Accessibility**: WCAG 2.2 AA compliance, which is now the baseline for UK Equality Act obligations and EU accessibility requirements for businesses serving EU customers
- **SEO**: Schema markup, proper semantic HTML, meta data, and a site structure that supports ranking
- **Security**: HTTPS, secure form handling, no exposed credentials or sensitive data
- **Mobile**: Mobile-first design tested across devices and screen sizes
