## The Problem

Fakenham Dog Field is a secure dog exercise field at Fakenham Racecourse. The owner needed a booking system that could handle payments and, critically, generate gate access codes automatically after payment.

The field is behind a Paxton-controlled gate. Without the right code, you cannot get in. Manual code distribution meant someone would need to be available every time a booking came in, which is not a sustainable model for a small business.

Off-the-shelf booking platforms were ruled out quickly. None of them could talk to a Paxton access control system. The ones that came closest would have required expensive customisation and still would not have done exactly what was needed.

## What We Built

A custom booking system deployed on Cloudflare Pages that handles everything without manual intervention.

**Availability and booking**: Customers browse available time slots, check what is free, and select a date and time. The calendar accounts for operating hours, existing bookings, and any blocked dates set by the owner.

**Payment**: Stripe handles checkout. No account creation required. The customer enters card details, pays, and the booking is confirmed.

**Access code generation**: Immediately after payment confirmation, the system calls the Paxton Cloud API to generate a time-limited access code. The code is valid only during the booked session window.

**Confirmation**: An automated email goes out with the booking details and the gate code. The whole process takes under two minutes.

**Admin dashboard**: The owner can view all bookings, manage availability, block dates, and see revenue reports. Full control, no technical knowledge required.

## The Integration That Made It Work

The hardware was installed and configured by [Surelock Security](https://www.surelocksecurity.co.uk) in Dereham, who handled the physical Paxton gate system on site. The software layer connects to the Paxton Cloud API so that booking confirmations automatically trigger valid, time-bound access credentials.

This combination of hardware expertise from Surelock and custom software integration from us meant the owner got a complete solution rather than two disconnected pieces.

## Business Impact

The system has been running reliably since launch. The owner has no manual tasks in the booking process. Customers can book and arrive without anyone else needing to be involved. The automated access codes work every time.
